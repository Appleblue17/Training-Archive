A. Ad-hoc Newbie

题目描述

Yuki gives you a sequence of n n n positive integers f 1 , … , f n f_1, \ldots, f_n f1​,…,fn​, where for each i \boldsymbol{i} i, 1 ≤ f i ≤ i \boldsymbol{1 \leq f_i \leq i} 1≤fi​≤i holds. She wants you to construct an n n n-ordered square matrix A A A such that:

It can be proven that for any valid f 1 , … , f n f_1, \ldots, f_n f1​,…,fn​, a solution always exists.

Recall that the mex ⁡ \operatorname{mex} mex of a sequence b 1 , … , b m b_1, \ldots, b_m b1​,…,bm​ is the smallest non-negative integer x x x such that x x x does not appear in b b b.

输入描述

Each test contains multiple test cases. The first line of input contains a single integer t t t ( 1 ≤ t ≤ 2 ⋅ 10 4 1 \leq t \leq 2\cdot 10^4 1≤t≤2⋅104) --- the number of test cases. The description of the test cases follows.

The first line contains a single integer n n n ( 1 ≤ n ≤ 1    414 1 \leq n \leq 1\;414 1≤n≤1414), denoting the length of the sequence.

The second line contains n n n integers f 1 , … , f n f_1, \ldots, f_n f1​,…,fn​ ( 1 ≤ f i ≤ i 1 \leq f_i \leq i 1≤fi​≤i), describing the given sequence.

It is guaranteed that the sum of n 2 n^2 n2 over all test cases does not exceed 2 ⋅ 10 6 2\cdot 10^6 2⋅106.

输出描述

For each test case, output n n n lines, in which the i i i-th line contains n n n non-negative integers A i , 1 , A i , 2 , … , A i , n A_{i,1}, A_{i,2}, \ldots, A_{i,n} Ai,1​,Ai,2​,…,Ai,n​ in the range [ 0 , n ] [0, n] [0,n].
