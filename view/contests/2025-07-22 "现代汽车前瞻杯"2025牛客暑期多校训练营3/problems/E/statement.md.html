E. Equal

题目描述

Yuki gives you a sequence of positive integers a 1 , … , a n a_1, \ldots, a_n a1​,…,an​ of length n n n, you can perform the following two operations any number of times:

  • Choose positive integers i , j , d i, j, d i,j,d such that 1 ≤ i < j ≤ n 1 \leq i < j \leq n 1≤i<j≤n and d ∣ a i , d ∣ a j d \mid a_i, d \mid a_j d∣ai​,d∣aj​, then update a i ← a i d a_i \gets \frac{a_i}{d} ai​←dai​​ while a j ← a j d a_j \gets \frac{a_j}{d} aj​←daj​​;
  • Choose positive integers i , j , d i, j, d i,j,d such that 1 ≤ i < j ≤ n 1 \leq i < j \leq n 1≤i<j≤n, then update a i ← a i ⋅ d a_i \gets a_i \cdot d ai​←ai​⋅d while a j ← a j ⋅ d a_j \gets a_j \cdot d aj​←aj​⋅d.

Determine whether it is possible to make a 1 = a 2 = … = a n a_1 = a_2 = \ldots = a_n a1​=a2​=…=an​ after several operations.

输入描述

Each test contains multiple test cases. The first line of input contains a single integer t t t ( 1 ≤ t ≤ 10 5 1 \leq t \leq 10^5 1≤t≤105) --- the number of test cases. The description of the test cases follows.

The first line contains an integer n n n ( 1 ≤ n ≤ 10 6 1 \leq n \leq 10^6 1≤n≤106), representing the length of the sequence.

The second line contains n n n integers a 1 , … , a n a_1, \ldots, a_n a1​,…,an​ ( 1 ≤ a i ≤ 5 ⋅ 10 6 1 \leq a_i \leq 5\cdot 10^6 1≤ai​≤5⋅106), describing the given sequence.

It is guaranteed that the sum of n n n across all test cases does not exceed 2 ⋅ 10 6 2\cdot 10^6 2⋅106.

输出描述

For each test case, print " YES \texttt{YES} YES" (without quotes) if it's possible to make all elements in a a a equal after several operations, and " NO \texttt{NO} NO" (without quotes) otherwise.

You can output the answer in any case (upper or lower). For example, the strings " y E s \tt{yEs} yEs", " y e s \tt{yes} yes", " Y e s \tt{Yes} Yes", and " Y E S \tt{YES} YES" will be recognized as positive responses.

备注

In the first test case, since n = 1 n = 1 n=1, all numbers are already the same, so the answer is " YES \texttt{YES} YES".

In the second test case, it can be shown that no matter what operation we perform, a 1 a_1 a1​ cannot be made equal to a 2 a_2 a2​.

In the third test case, you can choose i = 2 i = 2 i=2, j = 3 j = 3 j=3, d = 3 d = 3 d=3 and perform the first operation, transforming the original sequence into [ 1 , 1 , 1 ] [1, 1, 1] [1,1,1], where all numbers are the same; thus, the output is " YES \texttt{YES} YES".
