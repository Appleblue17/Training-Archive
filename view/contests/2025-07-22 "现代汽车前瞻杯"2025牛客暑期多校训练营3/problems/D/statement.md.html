D. Distant Control

题目描述

You have n n n robot friends, arranged in a row, labeled from left to right as 1 , 2 , … , n 1, 2, \ldots, n 1,2,…,n. Initially, some robots are turned off, while others are turned on.

Your phone can distantly control the power state of the robots, but unfortunately, it is not always very accurate. Specifically, there is a constant  1 ≤ a ≤ n − 1 1 \leq a \leq n - 1 1≤a≤n−1, and you can only perform the following two types of operations with your phone:

  • Select a a a robots with consecutive labels which are all turned on and turn them all off.
  • Select a + 1 a + 1 a+1 robots with consecutive labels which are all turned off and turn them all on.

You want to determine the maximum possible number of robots that are on after several (possibly, 0 0 0) operations.

输入描述

Each test contains multiple test cases. The first line of input contains a single integer t t t ( 1 ≤ t ≤ 4 ⋅ 10 4 1 \leq t \leq 4\cdot 10^4 1≤t≤4⋅104) --- the number of test cases. The description of the test cases follows.

The first line of each test case contains two integers n n n and a a a ( 2 ≤ n ≤ 2 ⋅ 10 5 2 \leq n \leq 2\cdot 10^5 2≤n≤2⋅105, 1 ≤ a ≤ n − 1 1 \leq a \leq n - 1 1≤a≤n−1).

The second line of each test case contains a string s s s of length n n n consisting only of 01 \tt{01} 01, describing the initial power states of the robots. Specifically, the i i i-th robot is initially turned on if and only if s i = 1 s_i = \tt{1} si​=1.

It is guaranteed that the sum of n n n over all test cases does not exceed 4 ⋅ 10 5 4\cdot 10^5 4⋅105.

输出描述

For each test case, output a single line containing an integer: the maximum possible number of robots that are on.

示例1

输入

4
3 1
001
4 3
0101
5 2
10110
10 4
1011010001

输出

3
2
5
5

备注

In the first test case, it is possible to perform a single operation to turn all robots on: select the robots labeled 1 , 2 1, 2 1,2, which are both off, and turn them on.

In the second test case, no operations can be performed, so the answer is the number of robots that are initially turned on, which is 2 2 2.
