C. Capella

题目描述

A string S S S consisting only of lowercase English letters is called Capella-like if and only if it satisfies all of the following conditions:

  • The number of letters that appear an odd number of times in  S S S is odd;
  • The number of letters that appear a non-zero even number of times in  S S S is even.

It can be shown that the string " capella \texttt{capella} capella" is Capella-like. However, the string " arcaea \texttt{arcaea} arcaea" is not Capella-like since there are 4 4 4 letters that appear an odd number of times, and " lowiro \texttt{lowiro} lowiro" is not Capella-like either since there is 1 1 1 letter that appears a non-zero even number of times and 4 4 4 letters that appear an odd number of times.

Now Yuki has a string S S S of length n n n consisting only of lowercase English letters. Yuki performs q q q operations on the string S S S: the i i i-th operation gives a position p i p_i pi​ ( 1 ≤ p i ≤ n 1 \leq p_i \leq n 1≤pi​≤n) and a lowercase letter c i c_i ci​, modifying the character at the p i p_i pi​-th position of S S S, denoted as S p i S_{p_i} Spi​​, to c i c_i ci​. You need to find the length of the longest substring of S S S that is Capella-like before all the q q q operations and after each operation.

Recall that a string S ′ S' S′ is a substring of S S S if and only if S ′ S' S′ can be obtained by removing a possibly empty prefix and suffix from S S S.

输入描述

The first line contains two integers n n n and q q q ( 1 ≤ n ≤ 2 ⋅ 10 5 1 \leq n \leq 2 \cdot 10^5 1≤n≤2⋅105, 1 ≤ q ≤ 2 ⋅ 10 5 1 \leq q \leq 2 \cdot 10^5 1≤q≤2⋅105).

The second line contains a string S S S of length n n n consisting only of lowercase English letters.

Each of the next q q q lines contains an integer p i p_i pi​ ( 1 ≤ p i ≤ n 1 \leq p_i \leq n 1≤pi​≤n) and a lowercase English letter c i c_i ci​, representing an operation.

输出描述

Output q + 1 q + 1 q+1 lines. Each line contains an integer representing the length of the longest substring of S S S that is Capella-like before all q q q operations and after each operation.

示例1

输入

6 4
ababab
3 c
6 a
4 c
2 a

输出

1
5
5
5
1
