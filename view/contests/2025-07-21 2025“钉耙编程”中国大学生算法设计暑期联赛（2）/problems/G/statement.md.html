G. 计数

Problem Description

给定一个长度为 n n n 的正整数序列 a 1 , a 2 , ⋯   , a n a_1, a_2, \cdots, a_n a1​,a2​,⋯,an​，以及一个正整数 R R R。

请你求出，有多少种长度为 n n n 的正整数序列 b 1 , b 2 , ⋯   , b n b_1, b_2, \cdots, b_n b1​,b2​,⋯,bn​，满足：

  • 对于任意 1 ≤ i ≤ n 1 \leq i \leq n 1≤i≤n，有 a i ≤ b i ≤ R a_i \leq b_i \leq R ai​≤bi​≤R。
  • 对于任意 1 ≤ i < n 1 \leq i < n 1≤i<n，有 b i ≥ b i + 1 b_i \geq b_{i + 1} bi​≥bi+1​。

答案对 10 9 + 7 10^9 + 7 109+7 取模。

Input

每个测试点中包含多组测试数据。输入的第一行包含一个正整数 T ( 1 ≤ T ≤ 10 ) T(1 \leq T \leq 10) T(1≤T≤10)，表示数据组数。对于每组测试数据：

第一行两个正整数 n , R ( 1 ≤ n ≤ 5 × 10 3 , 1 ≤ R ≤ 10 9 ) n, R(1 \leq n \leq 5 \times 10^3, 1 \leq R \leq 10^9) n,R(1≤n≤5×103,1≤R≤109)，分别表示序列长度与限制条件。

第二行 n n n 个正整数 a 1 , a 2 , ⋯   , a n ( 1 ≤ a i ≤ R ) a_1, a_2, \cdots, a_n(1 \leq a_i \leq R) a1​,a2​,⋯,an​(1≤ai​≤R)，表示序列 a a a。

保证所有测试数据中 n n n 之和不超过 5.2 × 10 3 5.2 \times 10^3 5.2×103。

Output

对于每组测试数据：输出一行一个整数，表示答案对 10 9 + 7 10^9 + 7 109+7 取模后的值。

Sample Input

2
5 5
3 1 3 4 4
4 1000
1 1 1 1

917124963
