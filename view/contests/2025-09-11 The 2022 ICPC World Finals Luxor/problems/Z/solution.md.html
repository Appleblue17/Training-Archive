Part 3

幅角信息并不好利用（因为知道的是实际值模 6 6 6 的值）。回到原来的模型。假设我们先根据 g 0 g_0 g0​ 的信息猜测一组 { a i } \{a_i\} {ai​}，考虑将某一个 a t a_t at​ 变成 2 a t 2a_t 2at​ 时，所有 2 n 2^n 2n 种组合构成的可重集 S S S 会发生什么变化：

  • 原来：设除了 a t a_t at​ 的所有数 2 n − 1 2^{n-1} 2n−1 种组合构成的可重集为 T T T，则 S = T × { 0 , a t } S = T \times \{0, a_t\} S=T×{0,at​}。

  • 现在： T T T 没有变化， S ′ = T × { 0 , 2 a t } = S ′ = T × { − a t , 0 } S' = T \times \{0, 2a_t\} = S' = T \times \{-a_t, 0\} S′=T×{0,2at​}=S′=T×{−at​,0}。

也就是说， S ′ S' S′ 可以由 S S S 中每个元素减去 a t a_t at​ 得到。也就是说， 如果将猜测的 { a i } \{a_i\} {ai​} 对应的 S S S 记作 R R R，设猜测的数里要变号的和为 x x x，那么 P P P 就是 R R R 每个数加上 x x x 得到的可重集。

知道了这个关键的性质后，整个问题只需要完成三步：

  1. 由猜测的 { a i } \{a_i\} {ai​} 计算出 R R R。
  2. 计算哪些 x x x 能够使 P P P 和 R R R 对上。
  3. 计算哪些 x x x 能够由猜测的 { a i } \{a_i\} {ai​} 组合出来，需要还原方案。

Part 3.3

如果不要求还原方案，直接按照 Part 3.1 的做法做即可。

要还原方案的话，似乎没有什么好做法。因为这里是三进制，如果某种 a i a_i ai​ 只有一个的话，构成的就不一定是线性空间，不能用线性基判。

最简单粗暴的做法是直接分层 DP 并记录每个状态有没有把当前层的 a t a_t at​ 取反。时间复杂度 O ( n 3 k ) O(n 3^k) O(n3k)。

综合以上做法，总时间复杂度为 O ( ( n + k ) 3 k ) O((n+k) 3^k) O((n+k)3k)。
