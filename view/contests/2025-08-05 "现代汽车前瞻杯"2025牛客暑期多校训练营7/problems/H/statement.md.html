H. Summit Down Side

题目描述

Note that the definition of mex ⁡ \textstyle \operatorname{mex} mex is different from other problems.

You are given a permutation of n \textstyle n n integers a 1 , a 2 , … , a n \textstyle a_1, a_2, \dots, a_n a1​,a2​,…,an​. Calculate the sum of mex ⁡ \textstyle \operatorname{mex} mex for all subarrays of a \textstyle a a.

The mex ⁡ \textstyle \operatorname{mex} mex of a non-empty set of integers S \textstyle S S is the smallest integer k \textstyle k k such that k ∉ S \textstyle k \notin S k∈/S and k > min ⁡ ( S ) \textstyle k > \min(S) k>min(S). For example, mex ⁡ ( { 1 , 3 , 5 } ) = 2 \textstyle \operatorname{mex}(\{1, 3, 5\}) = 2 mex({1,3,5})=2, mex ⁡ ( { 1 , 2 , 3 } ) = 4 \textstyle \operatorname{mex}(\{1, 2, 3\}) = 4 mex({1,2,3})=4.

A permutation of length n \textstyle n n is a sequence where each integer from 1 \textstyle 1 1 to n \textstyle n n appears exactly once.

输入描述

The first line contains a single integer t \textstyle t t ( 1 ≤ t ≤ 10 4 \textstyle 1 \le t \le 10^4 1≤t≤104), the number of test cases.

For each test case, the first line contains a single integer n \textstyle n n ( 1 ≤ n ≤ 5 × 10 5 \textstyle 1 \le n \le 5\times 10^5 1≤n≤5×105).

The second line contains n \textstyle n n distinct integers a 1 , a 2 , … , a n \textstyle a_1, a_2, \dots, a_n a1​,a2​,…,an​ ( 1 ≤ a i ≤ n \textstyle 1 \le a_i \le n 1≤ai​≤n).

It is also guaranteed that the sum of n \textstyle n n over all test cases will not exceed 5 × 10 5 \textstyle 5\times 10^5 5×105.

输出描述

For each test case, print a single integer in one line, representing the sum of mex ⁡ \textstyle \operatorname{mex} mex for all subarrays.
