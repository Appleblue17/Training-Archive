F. Random Segment Tree

题目描述

Define a random segment tree as a binary tree where each node represents a closed interval [ l , r ] [l, r] [l,r]:

  • If  l = r l = r l=r, then the node is a leaf node.
  • If l < r l < r l<r and the length of the interval ( r − l + 1 ) (r-l+1) (r−l+1) is even: let x = ⌊ l + r 2 ⌋ x = \left\lfloor \frac{l+r}{2} \right\rfloor x=⌊2l+r​⌋. The left child of the node represents [ l , x ] [l, x] [l,x], and the right child represents [ x + 1 , r ] [x+1, r] [x+1,r].
  • If l < r l < r l<r and the length of the interval ( r − l + 1 ) (r-l+1) (r−l+1) is odd: let x = l + r 2 x = \frac{l+r}{2} x=2l+r​. With a probability of 1 / 2 1/2 1/2, the left child represents [ l , x ] [l, x] [l,x] and the right child represents [ x + 1 , r ] [x+1, r] [x+1,r]; with a probability of 1 / 2 1/2 1/2, the left child represents [ l , x − 1 ] [l, x-1] [l,x−1] and the right child represents [ x , r ] [x, r] [x,r].

If [ x , y ] [x, y] [x,y] contains [ l , r ] [l, r] [l,r], the query ends. Otherwise, if the left child node intersects with [ x , y ] [x, y] [x,y], query the left child node; if the right child node intersects with [ x , y ] [x, y] [x,y], query the right child node. Given n n n, consider a random segment tree whose root node represents [ 1 , n ] [1, n] [1,n]. For 1 ≤ i ≤ 2 n 1 \leq i \leq 2n 1≤i≤2n, compute the expected number of intervals [ x , y ] [x, y] [x,y] where c o s t ( x , y ) \mathrm{cost}(x, y) cost(x,y) equals i i i, with the result modulo 998244353 998244353 998244353.
