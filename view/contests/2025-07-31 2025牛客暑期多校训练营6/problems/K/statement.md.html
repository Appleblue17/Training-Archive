K. Maximum GCD

题目描述

Given a sequence a a a of length n n n, you need to perform exactly one operation: select an interval and a non-negative integer X X X, and add X X X to all numbers in the interval, maximizing the GCD of all numbers in the sequence.  Output the maximun GCD，and if it is infinite, output 0 0 0.

输入描述

This problem contains multiple test cases.The first line contains a positive integer  T ( 1 ≤ T ≤ 10 5 ) T(1 \leq T \leq 10^5) T(1≤T≤105), representing the number of test cases.For each test case, the first input is a positive integer  n ( 1 ≤ n ≤ 10 5 ) n(1 \leq n \leq 10^5) n(1≤n≤105), representing the length of the sequence.Next, input  n n n positive integers, representing the sequence  a ( 1 ≤ a i ≤ 10 5 ) a(1 \leq a_i \leq 10^5) a(1≤ai​≤105).It is guaranteed that  ∑ n ≤ 2 ⋅ 10 5 \sum n \leq 2\cdot 10^5 ∑n≤2⋅105.

输出描述

For each test case, output a single line containing an integer, representing the answer.
