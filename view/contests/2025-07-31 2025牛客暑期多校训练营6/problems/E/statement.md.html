E. Array

题目描述

Given an array a a a of length n n n with all initial elements set to 0, and an array p p p of length n n n. You need to process q q q operations, which are of two types:

  • Update operation: Given four integers  o p t = 1 opt=1 opt=1,  l l l,  r r r,  x x x, for all  l ≤ i ≤ r l \le i \le r l≤i≤r, add  x x x to the element  a i a_i ai​.
  • Query operation: Given three integers  o p t = 2 opt=2 opt=2,  l l l,  r r r, compute  ∑ i = l r a p i \sum_{i=l}^{r} a_{p_{i}} ∑i=lr​api​​.

For each query operation, output the result of the query. This problem is strictly online, and for each operation, the input parameters l l l, r r r, x x x must be XORed with the last output answer (do not XOR before the first query operation).

输入描述

The first line contains two integers  n n n and  q q q ( 1 ≤ n , q ≤ 10 5 1 \leq n, q \leq 10^5 1≤n,q≤105), representing the length of the array  a a a and the number of operations.The second line contains  n n n integers, representing the array  p p p ( 1 ≤ p i ≤ n 1 \leq p_i \leq n 1≤pi​≤n).The next  q q q lines describe an operation, formatted as one of the two types mentioned above. For all decrypted inputs, there are  1 ≤ o p t ≤ 2 1 \le opt \le 2 1≤opt≤2,  1 ≤ l ≤ r ≤ n 1 \le l \le r \le n 1≤l≤r≤n, and  1 ≤ x ≤ 10 8 1 \le x \le 10^8 1≤x≤108.

输出描述

For each query operation, output one line with one integer, representing the result of the query.

38
