D. Beautiful Matrix

题目描述

A beautiful matrix A A A with n n n rows and n n n columns satisfies the following condition:

  • All elements are integers between  0 0 0 and  m m m.
  • A i , 1 = 0 A_{i, 1}=0 Ai,1​=0 for  1 ≤ i ≤ n 1 \leq i \leq n 1≤i≤n.
  • A i , j ≤ A i , j + 1 A_{i, j} \leq A_{i, j+1} Ai,j​≤Ai,j+1​ for  1 ≤ i ≤ n 1 \leq i \leq n 1≤i≤n and  1 ≤ j < n 1 \leq j \lt n 1≤j<n.
  • A i , j + A k , l ≤ A i , l + A k , j A_{i, j}+A_{k, l} \leq A_{i, l}+A_{k, j} Ai,j​+Ak,l​≤Ai,l​+Ak,j​ for every quadruple of integers  ( i , j , k , l ) (i,j,k,l) (i,j,k,l) such that  1 ≤ i < k ≤ n 1 \leq i \lt k \leq n 1≤i<k≤n and  1 ≤ j < l ≤ n 1 \leq j \lt l \leq n 1≤j<l≤n.

You are given two integers n n n and m m m, find the number, mod 998244353 998244353 998244353, of beautiful matrices.

输入描述

The first line contains two integers  n n n and  m m m ( 2 ≤ n ≤ 5 ⋅ 10 5 2 \le n \le 5\cdot 10^5 2≤n≤5⋅105,  1 ≤ m ≤ 5 ⋅ 10 5 1 \le m \le 5\cdot 10^5 1≤m≤5⋅105).

输出描述

Print the answer.
