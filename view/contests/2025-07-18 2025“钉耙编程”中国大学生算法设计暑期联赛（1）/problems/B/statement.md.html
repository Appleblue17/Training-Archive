B. 夜世界

Problem Description

你来到了夜世界，这是一个神奇的地方，这里有金矿、哥布林、时光钟楼。

很幸运，夜世界的领主赏赐给你 n n n 座排成一行的金矿，同时要求你在这里停留 m m m 天，每座金矿有一个属性 a i a_i ai​，代表该金矿一天产出的金币数量，但是，每座金矿中都潜伏着一只哥布林，每只哥布林都有一个属性 b i b_i bi​，代表这只哥布林的贪婪值。

每天夜晚，你将从第 1 1 1 座金矿走到第 n n n 座金矿，每走过一座金矿，以下两件事情依次发生：

  • 你将获得此座金矿当天产出的金币；
  • 潜伏在此座金矿中的哥布林露出了爪牙，向你索要与它贪婪值相等的金币，如果你没有这么多金币，那你只能将自己所有的金币都交给它，随后它看你穷得可怜，会放你离开。

受到神秘力量的影响，每天早晨都会发生以下某一事件：

  • 第 x x x 座金矿一天产出的金币数量变为 y y y，修改是持久的；
  • 第 x x x 座金矿中潜伏的哥布林的贪婪值变为 y y y，修改是持久的；
  • 时光钟楼显灵了！夜世界的状态被回溯到了第 x x x 天！这意味着 a , b a,b a,b 数组的状态将回退至与第 x x x 天夜晚一致，初始给出的 a , b a,b a,b 数组表示第 0 0 0 天夜晚的状态；
  • 有 k k k 座金矿中潜伏的哥布林变得穷凶极恶，当天夜晚，它们将不再向你索要与它们贪婪值相等的金币，若你目前拥有 s u m sum sum 枚金币，它们将向你索要 ⌈ s u m 2 ⌉ \lceil \frac{sum}{2} \rceil ⌈2sum​⌉ 枚金币，随后它们的神智清醒，不再穷凶极恶，但是你仍然很害怕，并希望知道当天夜晚你总共会交给哥布林多少金币，你需要打印这个值。

第一行包含一个整数 T T T，表示测试数据的组数， 1 ≤ T ≤ 10 1 \leq T \leq 10 1≤T≤10。

第三行包含 n n n 个整数 b 1 ,   b 2 ,   b 3 ,   … ,   b n    ( 0 ≤ b i ≤ 1 e 9 ) b_1, \thinspace b_2, \thinspace b_3, \thinspace \dots, \thinspace b_n \thickspace (0 \leq b_i \leq 1e9) b1​,b2​,b3​,…,bn​(0≤bi​≤1e9)，表示每座金矿中潜伏的哥布林的贪婪值；

接下来 m m m 行，第 i i i 行按如下格式给出一个操作：

  • 1    x    y :    1 \thickspace x \thickspace y: \thickspace 1xy: 第 x x x 座金矿一天产出的金币数量变为 y y y；
  • 2    x    y :    2 \thickspace x \thickspace y: \thickspace 2xy: 第 x x x 座金矿中潜伏的哥布林的贪婪值变为 y y y；
  • 3    x :    3 \thickspace x: \thickspace 3x: 夜世界的状态被回溯到了第 x    ( 0 ≤ x < i ) x \thickspace (0 \leq x < i) x(0≤x<i) 天；
  • 4    k :    4 \thickspace k: \thickspace 4k: 随后给出 k    ( k ≥ 0 ) k \thickspace (k \ge 0) k(k≥0) 个严格递增的金矿的位置，潜伏在这些金矿中的哥布林变得穷凶极恶。
