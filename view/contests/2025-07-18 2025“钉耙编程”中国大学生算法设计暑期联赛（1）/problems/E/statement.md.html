E. 传送门

Problem Description

在艾泽拉大陆上，有 n n n个被遗忘的遗迹。

冒险者小 S S S得知，在编号为 n n n 的遗迹中藏有传说中的宝藏，因此他决定从 1 1 1号遗迹出发，前往 n n n号遗迹。

大陆上有 m m m个传送门，每个传送门连接两个遗迹，并且可以双向通行。每个传送门属于一个魔法协会。

小 S S S可以向魔法协会支付一枚金币，使用该协会的传送门进行传送。在连续的传送中，若小 S S S使用的都是同一个协会的传送门，则只需要支付一枚金币。若小 S S S使用另一个协会的传送门，则需要重新支付一枚金币。

例如：小 S S S通过 A A A协会传送门到达某个遗迹，然后使用 B B B协会传送门前往下一个遗迹，则小 S S S需要向 B B B协会支付一枚金币，若小S之后想重新使用 A A A协会传送门，则还需向 A A A协会支付金币。

请帮助小 S S S设计一条传送路线，使得他从 1 1 1号遗迹到 n n n号遗迹需要支付的金币数量最少，并输出这个最小金币数。保证可以从1号遗迹到 n n n号遗迹。

Input

第一行输入一个整数 T T T（ 1 ≤ T ≤ 10 1 \le T \le 10 1≤T≤10），表示测试的总数。

第二行包含两个整数 n n n 和 m m m ，( 1 ≤ n ≤ 100000 1 \le n \le 100000 1≤n≤100000， 1 ≤ m ≤ 200000 1 \le m \le 200000 1≤m≤200000) 表示遗迹数量和传送门数量。

接下来 m m m 行，每行三个整数 u , v , c u, v, c u,v,c，( 1 ≤ u , v ≤ n 1 \le u, v \le n 1≤u,v≤n, 1 ≤ c ≤ 1000000 1 \le c \le 1000000 1≤c≤1000000) 表示遗迹 u u u 和 v v v 之间有一个协会 c c c 的传送门。

保证样例中 ∑ n ≤ 200000 , ∑ m ≤ 400000 \sum n \leq 200000, \sum m \leq 400000 ∑n≤200000,∑m≤400000 。

Output

每行输出一个整数，表示最小金币数。

Sample Input

1
4 3
1 2 1
2 3 3
3 4 1

Sample Output

3
