J. 中位数

Problem Description

对于一个长度为 L L L （ L L L 为奇数） 的数组 a a a，定义它的中位数 median \text{median} median（a） 为 a a a 中第 L + 1 2 \dfrac{L+1}2 2L+1​ 大的数。

现在给你一个长度为 n n n 的排列，对于每对满足 1 ≤ i ≤ j ≤ n 1\leq i\leq j\leq n 1≤i≤j≤n 且 j − i ≡ 0 ( m o d 2 ) j − i \equiv 0\pmod 2 j−i≡0(mod2) 的 （ i i i, j j j），你需要计算 i ∗ j ∗ median i*j * \text{median} i∗j∗median（ p [ i … j ] p[i\dots j] p[i…j]）。

输出所有值的总和。

Input

第一行输入一个整数 T T T （ 1 ≤ T ≤ 20 1 \le T \le 20 1≤T≤20），表示测试的总数。

对于每个测试用例，第一行输入一个整数 n n n（ 1 ≤ n ≤ 2000 1 \le n \le 2000 1≤n≤2000）。

接来下一行 n n n 个整数，表示 1 1 1 到 n n n 的排列。

Output

对于每个测试用例，输出一个数表示答案。

Sample Input

2
3
1 3 2
5
2 3 1 5 4

Sample Output

37
308
