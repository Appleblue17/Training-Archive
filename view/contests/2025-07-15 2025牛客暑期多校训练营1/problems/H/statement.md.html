H. Symmetry Intervals 2

题目描述

The time limit has been changed from 4 seconds to 2 seconds for upsolving.

Given a binary string S S S of length n n n, you need to answer q q q queries, where each query is in one of the following two types:

  1. Given two integers l , r l,r l,r ( 1 ≤ l ≤ r ≤ n 1 \le l \le r \le n 1≤l≤r≤n), flip the binary bit S i S_i Si​ for each i ∈ [ l , r ] i \in [l,r] i∈[l,r].
  2. Given three integers l , a , b l,a,b l,a,b ( 1 ≤ l ≤ n , 1 ≤ a , b ≤ n − l + 1 1 \le l \le n, 1 \le a,b \le n - l + 1 1≤l≤n,1≤a,b≤n−l+1), you need to find the number of intervals [ u , v ] [u,v] [u,v] ( 1 ≤ u ≤ v ≤ l 1 \le u \le v \le l 1≤u≤v≤l) such that S a + x − 1 = S b + x − 1 S_{a+x-1} = S_{b+x-1} Sa+x−1​=Sb+x−1​ holds for every integer x ∈ [ u , v ] x \in [u,v] x∈[u,v], which are called symmetry intervals.

输入描述

The first line contains two integers n n n and q q q ( 1 ≤ n , q ≤ 10 6 1\le n, q \le 10^6 1≤n,q≤106), indicating the length of the given string S S S and the number of queries.

The second line contains the given binary string S S S of length n n n.

The next q q q lines each contain a query, which is in one of the following two types:  · 1   l   r 1\ l\ r 1 l r ( 1 ≤ l ≤ r ≤ n 1 \le l \le r \le n 1≤l≤r≤n), indicating that for each binary bit S i S_i Si​ ( i ∈ [ l , r ] i \in [l,r] i∈[l,r]), flip the binary bit S i S_i Si​ for each i ∈ [ l , r ] i \in [l,r] i∈[l,r].  · 2   l   a   b 2\ l\ a\ b 2 l a b ( 1 ≤ l ≤ n , 1 ≤ a , b ≤ n − l + 1 1 \le l \le n, 1 \le a,b \le n - l + 1 1≤l≤n,1≤a,b≤n−l+1), indicating that you need to find the number of intervals [ u , v ] [u,v] [u,v] ( 1 ≤ u ≤ v ≤ l 1 \le u \le v \le l 1≤u≤v≤l) such that S a + x − 1 = S b + x − 1 S_{a+x-1} = S_{b+x-1} Sa+x−1​=Sb+x−1​ holds for every integer x ∈ [ u , v ] x \in [u,v] x∈[u,v].

It is guaranteed that the number of type-2 queries does not exceed 2   500 2\,500 2500.

For each type-2 query, output a line containing an integer, indicating the number of symmetry intervals.

说明

For the first query, S 3..6 = 0100 S_{3..6} = 0100 S3..6​=0100, S 5..8 = 0010 S_{5..8} = 0010 S5..8​=0010, the 2 2 2 symmetry intervals are [ 1 , 1 ] [1,1] [1,1] and [ 4 , 4 ] [4,4] [4,4].

After the second query, S S S becomes 1110111001 1110111001 1110111001.

For the third query, S 2..6 = 11011 S_{2..6} = 11011 S2..6​=11011, S 6..10 = 11001 S_{6..10} = 11001 S6..10​=11001, the 7 7 7 symmetry intervals are [ 1 , 1 ] [1,1] [1,1], [ 1 , 2 ] [1,2] [1,2], [ 1 , 3 ] [1,3] [1,3], [ 2 , 2 ] [2,2] [2,2], [ 2 , 3 ] [2,3] [2,3], [ 3 , 3 ] [3,3] [3,3], and [ 5 , 5 ] [5,5] [5,5].
