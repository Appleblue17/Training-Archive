A. All Kinds of Dice

题目描述

Define a ``die'' as any shape with at least one face such that each face shows a positive integer. When a die is rolled, one of its faces is selected uniformly at random. When two dice roll against each other, the die whose selected face shows a higher number earns 1 1 1 point; if both numbers are equal, each die earns 1 / 2 1/2 1/2 points. For dice D D D and D 0 D_0 D0​, define s c o r e ( D , D 0 ) \mathrm{score}(D, D_0) score(D,D0​) as the expected number of points D D D earns from a single roll against D 0 D_0 D0​.

Given a tournament of N N N vertices, please construct N N N dice D 1 , D 2 , … , D N D_1,D_2,\ldots,D_N D1​,D2​,…,DN​ such that

  • For every directed edge i → j i \to j i→j, s c o r e ( D i , D j ) > 1 / 2 \mathrm{score}(D_i,D_j) > 1/2 score(Di​,Dj​)>1/2.

Report if it is impossible to achieve the goal.

The first line of the input contains a single integer N N N ( 2 ≤ N ≤ 100 2 \le N \le 100 2≤N≤100), denoting the number of vertices in the tournament.

Then, N N N lines follow. The i i i-th line contains a binary string s i s_i si​.

It is guaranteed that exactly one of s i , j s_{i,j} si,j​ and s j , i s_{j,i} sj,i​ equals to ‘1’ \texttt{`1'} ‘1’ for any 1 ≤ i < j ≤ N 1 \le i < j \le N 1≤i<j≤N and s i , i = ‘0’ s_{i,i} = \texttt{`0'} si,i​=‘0’ for all 1 ≤ i ≤ N 1 \le i \le N 1≤i≤N.

输出描述

If there exists no solution, output − 1 -1 −1 in one line.

Otherwise, output N N N lines. The i i i-th line describes the i i i-th dice:\ ∙ \bullet ∙ The first integer k k k denotes the faces of the i i i-th ``die'';\ ∙ \bullet ∙ Then k k k integers v 1 , v 2 , … , v k v_1,v_2,\ldots,v_k v1​,v2​,…,vk​ ( 1 ≤ v i ≤ 10 9 1 \le v_i \le 10^9 1≤vi​≤109) follow, denoting the integers in each face.
