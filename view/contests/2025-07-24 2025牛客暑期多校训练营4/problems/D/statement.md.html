D. Determinant of 01-Matrix

题目描述

Given a positive integer N ≤ 10 9 N \le 10^9 N≤109, please construct a matrix A = ( a i , j ) 1 ≤ i , j ≤ n A=(a_{i,j})_{1 \le i,j \le n} A=(ai,j​)1≤i,j≤n​ that contains only 0 0 0s and 1 1 1s, satisfying\

  1. 1 ≤ n ≤ 200 1 \le n \le 200 1≤n≤200;
  2. d e t ( A ) = N \mathrm{det}(A) = N det(A)=N.

It is guaranteed that there exists a valid solution under the problem constraints.\

输入描述

The input contains only one positive integer N N N ( 1 ≤ N ≤ 10 9 1 \le N \le 10^9 1≤N≤109).

输出描述

The output should contain a first line with a positive integer n n n ( 1 ≤ n ≤ 200 1 \le n \le 200 1≤n≤200), indicating the size of the constructed matrix.

Next, output n n n lines, where the i i i-th line contains n n n integers a i , 1 , a i , 2 , … , a i , n a_{i,1}, a_{i,2},\ldots,a_{i,n} ai,1​,ai,2​,…,ai,n​ ( a i , j ∈ { 0 , 1 } a_{i,j} \in \{0,1\} ai,j​∈{0,1}), representing the elements of the i i i-th row of the matrix. If there are multiple solutions, any one of them can be output.\

输入

1

输出

1
1

输入
