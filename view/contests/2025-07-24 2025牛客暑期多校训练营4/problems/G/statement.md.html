G. Ghost in the Parentheses

题目描述

Alice has a valid parentheses sequence s s s and she wants to share it with Bob through a noisy channel. For each character, the channel independently transmits that character to Bob with a probability of 1 / 2 1/2 1/2, or transmits an indistinguishable character `?' to Bob with a probability of 1 / 2 1/2 1/2. For example, if Alice's valid parentheses sequence is s s s = (()()), Bob might receive the character sequences (?)()), ?(??)?, but cannot receive the character sequence ??(()). After receiving the message, Bob tries to reconstruct the valid parentheses sequence that Alice transmitted. However, Alice knows that in some cases, the reconstruction is not unique: for instance, when receiving the character sequence ?(??)?, there are two possible reconstructions: (()()) and ((())).
\

Please help Alice calculate the probability that the message Bob received can uniquely reconstruct the valid parentheses sequence. Output the answer modulo 998   244   353 998\,244\,353 998244353.

\

\

输入描述

The input consists of a single line containing a valid parentheses sequence s s s ( 2 ≤ ∣ s ∣ ≤ 10 6 2 \le |s| \le 10^6 2≤∣s∣≤106).

输出描述

Output a single integer indicating the probability that the message Bob received can uniquely reconstruct the valid parentheses sequence, modulo 998   244   353 998\,244\,353 998244353.

示例1

输入

()()

输出

249561089

说明

For the first example, the message Bob received can uniquely reconstruct the valid parentheses sequence if and only if the second character and the third character are not both lost. The probability of this event occurring is 3 4 ≡ 249   561   089 m o d     998   244   353 \frac{3}{4} \equiv 249\,561\,089 \mod 998\,244\,353 43​≡249561089mod998244353.
