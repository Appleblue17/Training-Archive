J. Journey Around the World

题目描述

There is a grid with n n n rows and n n n columns ( 3 ≤ n ≤ 200 3 \leq n \leq 200 3≤n≤200). For 1 ≤ i , j ≤ n 1 \leq i,j \leq n 1≤i,j≤n, there is an undirected edge between the cell in the i i i-th row and j j j-th column and the cell in the i i i-th row and ( j + 1 ) (j+1) (j+1)-th column (the ( n + 1 ) (n+1) (n+1)-th column is considered as the 1 1 1-st column), with a weight of either 1 1 1 or 2 2 2. For 1 ≤ i < n 1 \leq i < n 1≤i<n and 1 ≤ j ≤ n 1 \leq j \leq n 1≤j≤n, there is an undirected edge between the cell in the i i i-th row and j j j-th column and the cell in the ( i + 1 ) (i+1) (i+1)-th row and j j j-th column, also with a weight of either 1 1 1 or 2 2 2.

For 1 ≤ i ≤ n 1 \leq i \leq n 1≤i≤n, find the shortest path length from the cell in the 1 1 1-st row and i i i-th column to the cell in the n n n-th row and i i i-th column, passing through each column at least once.

输入描述

The first line contains a positive integer T T T representing the number of test cases. The sum of n n n across all test cases does not exceed 200 200 200.

For each test case, the first line contains a positive integer n n n.

The next n n n lines each contain n n n integers, where the j j j-th integer of the i i i-th line represents the weight of the edge between the cell in the i i i-th row and j j j-th column and the cell in the i i i-th row and ( j + 1 ) (j+1) (j+1)-th column.

The following n − 1 n-1 n−1 lines each contain n n n integers, where the j j j-th integer of the i i i-th line represents the weight of the edge between the cell in the i i i-th row and j j j-th column and the cell in the ( i + 1 ) (i+1) (i+1)-th row and j j j-th column.

输出描述

For each test, output a total of n n n lines, where the i i i-th line contains a single integer representing the shortest path length from the cell in the 1 1 1-st row and i i i-th column to the cell in the n n n-th row and i i i-th column, passing through each column at least once.
