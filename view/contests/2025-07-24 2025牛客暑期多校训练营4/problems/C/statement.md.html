C. Computational Geometry Problem

题目描述

Given integer N N N, M M M, and K K K, calculate the number of integer sequences A = [ A 1 , A 2 , … , A N M ] A = [A_1,A_2,\ldots,A_{NM}] A=[A1​,A2​,…,ANM​] satisfying the following constraint:

  1. 1 ≤ A i ≤ M 1 \le A_i \le M 1≤Ai​≤M and each number from 1 1 1 to M M M appears in A A A exactly N N N times;
  2. There do not exist integers 1 ≤ i < j < k < ℓ ≤ N M 1 \le i<j<k<\ell \le NM 1≤i<j<k<ℓ≤NM satisfying A i = A k A_i = A_k Ai​=Ak​, A j = A ℓ A_j = A_\ell Aj​=Aℓ​, and A i ≠ A j A_i \neq A_j Ai​=Aj​;
  3. The number of 1 ≤ i < N M 1 \le i < NM 1≤i<NM such that A i = A i + 1 A_i = A_{i+1} Ai​=Ai+1​ is K K K.

The answer might be enormous, and you should output the answer modulo 998   244   353 998\,244\,353 998244353.

The only line of the input contains three integers N N N , M M M and K K K ( 1 ≤ N , M ≤ 10 7 1 \le N,M \le 10^7 1≤N,M≤107, 0 ≤ K ≤ N M − 1 0 \le K \le NM-1 0≤K≤NM−1).
