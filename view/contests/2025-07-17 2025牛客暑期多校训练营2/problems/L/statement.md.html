L. Love Wins All

题目描述

It's a loving community!

There are n n n residents in the community, and each resident i   ( 1 ≤ i ≤ n ) i\ (1\leq i\leq n) i (1≤i≤n) in the community has a unique resident a i   ( 1 ≤ i ≤ n ) a_i\ (1\leq i\leq n) ai​ (1≤i≤n) in the community whom he/she loves so much. Each two residents love different residents. A resident can love himself / herself. It is guaranteed that n n n is even.

One day, a bad thing happens: They need to choose 2 2 2 residents to be forbidden to get married forever.

And to prevent such a thing from happening in the future, the rest n − 2 n-2 n−2 residents decide to get married as n 2 − 1 \frac{n}{2}-1 2n​−1 couples, each couple consisting of 2 2 2 persons (of course). It makes no sense that a couple consists of resident x x x and resident y y y while neither x x x loves y y y nor y y y loves x x x, so such a thing never happens.

So, as the planner, you need to figure out how you can arrange this. You want to know the number of different marriage plans. Two marriage plans are considered different if at least one of the following conditions is satisfied:

  • In one plan, a person i i i is married, and in the other, he/she is not.
  • In one plan, a person i i i is married to j j j, and in the other, he/she is not married to j j j.

As the number of plans can be quite enormous, output it modulo 998   244   353 998\ 244\ 353 998 244 353.

输入描述

Each test contains multiple test cases. The first line contains the number of test cases T   ( 1 ≤ T ≤ 10 4 ) T\ (1≤T≤10^4) T (1≤T≤104) .

Each test case consists of two lines.

The first line contains 1 1 1 integer n   ( 4 ≤ n ≤ 5 × 10 5 ) n\ (4\leq n\leq 5\times 10^5) n (4≤n≤5×105), the number of residents in the community. It's guaranteed that n n n is even.

The second line contains n n n integers a 1 , a 2 , … , a n   ( 1 ≤ a i ≤ n ) a_1,a_2,\dots,a_n\ (1\leq a_i\leq n) a1​,a2​,…,an​ (1≤ai​≤n), where a i a_i ai​ represents the one that the resident i i i loves. It is guaranteed that if i ≠ j   ( 1 ≤ i , j ≤ n ) i\neq j\ (1\leq i,j\leq n) i=j (1≤i,j≤n), a i ≠ a j a_i\neq a_j ai​=aj​ .

It is guaranteed that ∑ n \sum n ∑n over all test cases in one test will not exceed 5 × 10 5 5\times 10^5 5×105 .

输出描述

For each test case, output 1 1 1 integer: the number of different marriage plans modulo 998   244   353 998\ 244\ 353 998 244 353.
