M. Miscalculated Triangles

题目描述

In his homework, Sean ran into this problem:

--- How many different shapes of triangles are there in a 2D-plane, with lengths of its sides in { 1 , 2 , … , s } \{1,2,\dots,s\} {1,2,…,s} and perimeter no longer than l l l ? Two triangles are considered to be in the same shape if they can completely overlap using only translation and rotation. Note that flips are not allowed. So for Δ A B C \Delta ABC ΔABC and Δ A ′ B ′ C ′ \Delta A'B'C' ΔA′B′C′ ( A , B , C A,B,C A,B,C[ and ]{style="text-wrap-mode:wrap;"} A ′ , B ′ , C ′ A',B',C' A′,B′,C′[ are listed anti-clockwise.]{style="text-wrap-mode:wrap;"}), if A B = 2 , B C = 3 , C A = 4 AB=2,BC=3,CA=4 AB=2,BC=3,CA=4 and A ′ B ′ = 2 , A ′ C ′ = 4 , C ′ A ′ = 3 A'B'=2,A'C'=4,C'A'=3 A′B′=2,A′C′=4,C′A′=3, they are not considered to be in the same shape.

Sean likes to consider problems in the binary system, so he uses the binary representation of numbers. He iterated over all possible ( a , b , c ) (a,b,c) (a,b,c) trying to find the answer, so what he wanted is the number of triplets ( a , b , c ) (a,b,c) (a,b,c) such that:

  • 1 ≤ a , b , c ≤ s 1\leq a,b,c\leq s 1≤a,b,c≤s and are all integers.
  • a + b > c , a + c > b , b + c > a a+b\gt c, a+c\gt b, b+c\gt a a+b>c,a+c>b,b+c>a - a + b + c ≤ l a+b+c\leq l a+b+c≤l

Then, he chose from these triplets such that each triplet chosen represents a different triangle. However, Sean was so bad at math that when he calculated a + b + c a+b+c a+b+c , he totally forgot about the carries, and therefore got the result of a ⊕ b ⊕ c a⊕b⊕c a⊕b⊕c, which is the bitwise exclusive OR (XOR) sum of a , b a,b a,b and c c c .

Despite this mistake, Sean wants to know if he makes other mistakes, so he asks you about the answer if the third condition is a ⊕ b ⊕ c ≤ l a⊕b⊕c\leq l a⊕b⊕c≤l rather than a + b + c ≤ l a+b+c\leq l a+b+c≤l. Can you help him out with this?

Since the answer can be enormous, output it modulo 998   244   353 998\ 244\ 353 998 244 353 .

Each test contains multiple test cases. The first line contains the number of test cases T   ( 1 ≤ T ≤ 10 4 ) T\ (1≤T≤10^4) T (1≤T≤104) .

The line contains 2 2 2 strings s l , s s   ( ∣ s l ∣ ≤ 10 5 , ∣ s s ∣ ≤ 10 5 ) s_l, s_s\ (|s_l|\leq 10^5,|s_s|\leq 10^5) sl​,ss​ (∣sl​∣≤105,∣ss​∣≤105) , the binary representation of the integers l l l and s s s . It is guaranteed that l > 0 l\gt 0 l>0 and s > 0 s\gt 0 s>0, and the first character of string s s s_s ss​ and s l s_l sl​ is always 1 1 1 .

It is guaranteed that ∑ ∣ s l ∣ \sum|s_l| ∑∣sl​∣ over all test cases in one test will not exceed 5 × 10 5 5\times 10^5 5×105 and ∑ ∣ s s ∣ \sum|s_s| ∑∣ss​∣ over all test cases in one test will not exceed 5 × 10 5 5\times 10^5 5×105.

输出描述

For each test case, output 1 1 1 integer: the number of triangles in different shapes modulo 998   244   353 998\ 244\ 353 998 244 353 .
