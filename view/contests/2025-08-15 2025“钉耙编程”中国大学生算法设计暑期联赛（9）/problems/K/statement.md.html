K. 抽象代数

Problem Description

在 U = {   1 , 2 , ⋯   , n   } U= \set{ 1, 2, \cdots, n } U={1,2,⋯,n} 上定义有结合律的二元运算 ⊕ \oplus ⊕ 满足 ∀ a , b ∈ U , a ⊕ b ∈ { a , b } \forall a,b\in U, a\oplus b\in \{a,b\} ∀a,b∈U,a⊕b∈{a,b}。

对于一个运算 ⊕ \oplus ⊕，称数 x x x 是交换子当且仅当 ∀ y ∈ U , x ⊕ y = y ⊕ x \forall y\in U, x\oplus y=y\oplus x ∀y∈U,x⊕y=y⊕x。

已知 m m m 个形如 a ⊕ b = c a\oplus b=c a⊕b=c 的等式（ 1 ≤ a , b ≤ K ≤ 17 1\le a,b\le K\le 17 1≤a,b≤K≤17， c ∈ { a , b } c\in \{a,b\} c∈{a,b}），对于所有可能的 ⊕ \oplus ⊕ 求 2 交换子个数 2^{\text{交换子个数}} 2交换子个数 和。

Input

首先输入三个自然数 n , m , K n,m,K n,m,K（ 2 ≤ n ≤ 10 5 2\le n\le 10^5 2≤n≤105， 0 ≤ m ≤ K 2 0\le m \le K^2 0≤m≤K2， 1 ≤ K ≤ 17 1\le K\le 17 1≤K≤17），分别表示元素个数、已知等式个数和已知等式的数字范围。

之后有 m m m 行，每行有三个正整数 a , b , c a,b,c a,b,c（ 1 ≤ a , b ≤ K 1\le a,b\le K 1≤a,b≤K， c ∈ { a , b } c\in \{a,b\} c∈{a,b}），表示一个等式 a ⊕ b = c a\oplus b=c a⊕b=c。

Output

输出一行一个自然数，表示答案除以 998244353 998244353 998244353 的余数。

Sample Input

3 3 3
1 2 1
2 3 2
2 1 2

Sample Output

3
