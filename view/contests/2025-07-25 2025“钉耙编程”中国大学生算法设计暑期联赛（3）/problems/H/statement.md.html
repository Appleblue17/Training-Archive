H. 01环

Problem Description

给定一个由字符 '0' 和 '1' 组成的长度为 N N N 的环形字符串，其中 N N N 是一个偶数。你可以通过以下两种操作将这个环调整为 01 01 01 交替的形式（即相邻字符不相同，如 "0101..." 或 "1010..."）：

  • 交换操作：选择任意一个位置 i i i，交换 i i i 和   ( i   m o d   N ) + 1 ~(i \bmod N) + 1  (imodN)+1 位置的字符（即相邻位置交换，环的首尾也视为相邻）。
  • 翻转操作：选择任意一个位置 i i i，将该位置的字符取反（'0' 变 '1'，'1' 变 '0'）。

你的任务是计算出将给定环形字符串转换为 01 01 01 交替环所需的最少操作次数。

Input

第一行包含一个整数 T T T（ 1 ≤ T ≤ 10 4 1 \leq T \leq 10^4 1≤T≤104），表示测试用例的数量。

每组测试用例包含两行：

  • 第一行为一个偶数 N N N（ 2 ≤ N ≤ 10 6 2 \leq N \leq 10^6 2≤N≤106），表示环形字符串的长度。
  • 第二行为一个长度为 N N N 的字符串 s s s，仅由字符 '0' 和 '1' 组成。

保证所有测试用例的 N N N 之和不超过 10 6 10^6 106。

Output

对于每组测试用例，输出一行一个整数，表示所需的最少操作次数。

Sample Input

2
4
0110
6
000000

Sample Output

1
3
