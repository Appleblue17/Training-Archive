J. 坚船利炮

Problem Description

你是一名舰长，你的战舰由 n n n 个舱室组成，并另有 n − 1 n-1 n−1 条通道，每条通道连接一对舱室；通过若干通道，任意一对舱室都互相可达（可以将战舰看作一棵树）。

在战斗中，由于海浪汹涌，船体颠簸，你和你的敌人都只能大致地瞄准对方：每当你的战舰被击中时，一条未被打断的通道将被打断，且可以认为每条未被打断的通道之间被（选中）打断的概率相等。
