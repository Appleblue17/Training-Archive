G. 满员电车

Problem Description

小 Q 每个工作日都需要搭乘电车进行通勤。仅有的一条双向电车线路包含 n n n 个站台，沿途依次编号为 1 , 2 , … , n 1,2,\dots,n 1,2,…,n， i i i 号站台与 1 号站台的距离为 d i d_i di​。一共有两类电车：由 1 号站台开往 n n n 号站台的正向电车以及由 n n n 号站台开往 1 号站台的逆向电车。

每天从早到晚一共有 m m m 个班次的正向电车，第 i i i 个班次的正向电车固定在时刻 a i a_i ai​ 从 1 号站台开出，在 a i + d j a_i+d_j ai​+dj​ 时刻停靠 j j j（ 1 ≤ j ≤ n 1\leq j\leq n 1≤j≤n）号站台。换句话说，乘客能在 j j j 号站台上车当且仅当他在时刻 a i + d j a_i+d_j ai​+dj​ 位于 j j j 号站台。你可以认为恰好在时刻 a i + d j a_i+d_j ai​+dj​ 在 j j j 号站台下车的乘客也可以赶上这趟换乘电车。

同理，每天从早到晚一共有 p p p 个班次的逆向电车，第 i i i 个班次的逆向电车固定在时刻 b i b_i bi​ 从 n n n 号站台开出，在 b i + d n − d j b_i+d_n-d_j bi​+dn​−dj​ 时刻停靠 j j j（ 1 ≤ j ≤ n 1\leq j\leq n 1≤j≤n）号站台。

每天早晨，小 Q 将来到家旁边的 S S S 号站台，搭乘电车到达公司旁边的 T T T（ S < T S < T S<T）号站台。早高峰的电车非常拥挤，但是规律可循，对于每个班次的电车，都存在一个区间 [ l , r ] [l,r] [l,r]，表示当且仅当这班电车停靠 k k k（ l ≤ k ≤ r l\leq k\leq r l≤k≤r）号站台时是不拥挤的，此时小 Q 才可以上车。但是只要小 Q 上了车，就能在任意一站下车，无论是否拥挤。

小 Q 可以通过换乘多趟电车的方式辗转完成通勤。请写一个程序找到最优的通勤路线，使得通勤时间最少。在这里，通勤时间定义为小 Q 最终在 T T T 号站台下车的时刻减去小 Q 第一次在 S S S 号站台上车的时刻。

Input

第一行包含一个正整数 T T T（ 1 ≤ T ≤ 300 1\leq T\leq 300 1≤T≤300），表示测试数据的组数。

每组数据第一行包含四个正整数 n , m , p , q n,m,p,q n,m,p,q（ 2 ≤ n ≤ 200   000 2\leq n\leq 200\,000 2≤n≤200000, 1 ≤ m , p , q ≤ 200   000 1\leq m,p,q\leq 200\,000 1≤m,p,q≤200000），分别表示站台、正向电车、逆向电车以及询问的数量。

第二行包含 n n n 个整数 d 1 , d 2 , … , d n d_1,d_2,\dots,d_n d1​,d2​,…,dn​（ 0 ≤ d i ≤ 10 8 0\leq d_i\leq 10^8 0≤di​≤108, d 1 = 0 d_1=0 d1​=0, d i < d i + 1 d_i < d_{i+1} di​<di+1​），分别表示每个站台到 1 号站台的距离。

接下来 m m m 行，每行三个整数 a i , l i , r i a_i,l_i,r_i ai​,li​,ri​（ 0 ≤ a i ≤ 10 8 0\leq a_i\leq 10^8 0≤ai​≤108, a i < a i + 1 a_i < a_{i+1} ai​<ai+1​, 1 ≤ l i ≤ r i ≤ n 1\leq l_i\leq r_i\leq n 1≤li​≤ri​≤n），依次描述每个班次的正向电车。

接下来 p p p 行，每行三个整数 b i , l i , r i b_i,l_i,r_i bi​,li​,ri​（ 0 ≤ b i ≤ 10 8 0\leq b_i\leq 10^8 0≤bi​≤108, b i < b i + 1 b_i < b_{i+1} bi​<bi+1​, 1 ≤ l i ≤ r i ≤ n 1\leq l_i\leq r_i\leq n 1≤li​≤ri​≤n），依次描述每个班次的逆向电车。

接下来 q q q 行，每行两个正整数 S i , T i S_i,T_i Si​,Ti​（ 1 ≤ S i < T i ≤ n 1\leq S_i < T_i\leq n 1≤Si​<Ti​≤n），分别表示每个询问的起点站与终点站。

输入数据保证 ∑ n ≤ 1   000   000 \sum n\leq 1\,000\,000 ∑n≤1000000, ∑ m ≤ 1   000   000 \sum m\leq 1\,000\,000 ∑m≤1000000, ∑ p ≤ 1   000   000 \sum p\leq 1\,000\,000 ∑p≤1000000 且 ∑ q ≤ 1   000   000 \sum q\leq 1\,000\,000 ∑q≤1000000。

Output

对于每个询问输出一行一个整数，即从 S i S_i Si​ 到 T i T_i Ti​ 的最少通勤时间，若无解请输出 " -1 \texttt{-1} -1"。
