G. Line of Sight

题目描述

::::::: background-info In 2024, team "Moon Cat" trained hundreds of hours together, walked through a lot of places, ended with regret of loss, but experienced many good things and memories, unpredicted and sparkling in mind as if a dream.

SATSKY remembers, whenever the team was heading to somewhere, in the streets or metro station or anywhere, it often occurs that laurxh "walks away without looking back, as if he had made some determination"(goes far ahead), while Genius_dream concentrates on her phone and drops too far behind. In this case, SATSKY prefers to track the middle point of two teammates' locations, or at least find a place where he can see his teammates both, especially when around the corner and they can't see each other.

Now SATSKY transformed this into a programming geometry problem, in commemoration of the memorable year with them, the treasure of his life.

Formally:

Give you an n \textstyle n n-sided polygon P \textstyle P P, of which each point P 1 ∼ n \textstyle P_{1 \sim n} P1∼n​ is of integer coordinates and given in counterclockwise order;

And give you two more integer points A \textstyle A A and B \textstyle B B, inside the polygon (boundary not included) guaranteed;

Call "point X \textstyle X X can be seen from point Y \textstyle Y Y (in P \textstyle P P)" if and only if the whole segment connecting two points, except two endpoints, is inside P \textstyle P P strictly (and doesn't intersect with any of edge of P \textstyle P P naturally) ;

You need to find out:

  • For each point of polygon ( i \textstyle i i from 1 \textstyle 1 1 to n \textstyle n n), can it ( P i \textstyle P_i Pi​) be seen from point A \textstyle A A ?
  • For each point of polygon ( i \textstyle i i from 1 \textstyle 1 1 to n \textstyle n n), can it ( P i \textstyle P_i Pi​) be seen from point B \textstyle B B ?
  • Is there a point C \textstyle C C inside the polygon (boundary not included) to see A \textstyle A A and B \textstyle B B simultaneously ?

:::::::

输入描述

The first line contains the number of test cases t \textstyle t t ( 1 ≤ t ≤ 2 × 10 5 \textstyle 1 \leq t \leq 2 \times 10^5 1≤t≤2×105). The description of the test cases follows.

The first line contains two integers n \textstyle n n ( 3 ≤ n ≤ 2 × 10 5 \textstyle 3 \leq n \leq 2 \times 10^5 3≤n≤2×105), indicating the number of points of P \textstyle P P;

The second line contains four integers x A , y A , x B , y B \textstyle x_A,y_A,x_B,y_B xA​,yA​,xB​,yB​ , representing the x and y coordinates of point A \textstyle A A and B \textstyle B B;

Then following n \textstyle n n lines, the i \textstyle i i-th line contains two integers x P i , y P i \textstyle x_{P_i},y_{P_i} xPi​​,yPi​​ , representing the x and y coordinates of point P i \textstyle P_i Pi​.

It's guaranteed that:

  • P 1 ∼ n \textstyle P_{1 \sim n} P1∼n​ are given in counterclockwise order and must form a polygon;
  • For A \textstyle A A, B \textstyle B B and each point of P \textstyle P P ( P 1 ∼ n \textstyle P_{1 \sim n} P1∼n​), all the coordinates satisfy ∣ x ∣ , ∣ y ∣ ≤ 10 9 \textstyle \lvert x \rvert,\lvert y \rvert \leq 10^9 ∣x∣,∣y∣≤109;
  • The sum of n \textstyle n n in each test file doesn't exceed 10 6 \textstyle 10^6 106 ( ∑ n ≤ 10 6 \textstyle \sum n \leq 10^6 ∑n≤106).

Some emphasis:

  • Adjacent edges of P \textstyle P P may be collinear, so are not adjacent pairs;
  • Any three points within A \textstyle A A, B \textstyle B B and P 1 ∼ n \textstyle P_{1 \sim n} P1∼n​ may be on the same straight line, A \textstyle A A and B \textstyle B B may even coincide;
  • The coordinates of C \textstyle C C you choose can be decimals.

输出描述

For each test case, answer it on three lines:

In the first line, output n \textstyle n n integers separated by spaces: the i \textstyle i i-th integer should be 1 if P i \textstyle P_i Pi​ can be seen from point A \textstyle A A, or 0 if it can't;

In the second line, output n \textstyle n n integers separated by spaces: the i \textstyle i i-th integer should be 1 if P i \textstyle P_i Pi​ can be seen from point B \textstyle B B, or 0 if it can't;

In the third line, if there exists a point C \textstyle C C meeting the requirements above, output a string "Yes" (without quotes); otherwise output a string "No" (without quotes).
