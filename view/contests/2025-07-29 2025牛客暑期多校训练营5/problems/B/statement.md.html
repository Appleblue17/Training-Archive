Note that uppercase and lowercase letters are treated as distinct.

输出描述

Output a single integer, the number of distinct strings that can be formed by concatenating [[ k k k][[[]{.strut style="height:0.69444em;vertical-align:0em;"}[k]{.mord .mathdefault style="margin-right:0.03148em;"}]{.base}]{.katex-html aria-hidden="true"}] substrings (selected in order from the 1st to the k-th operation) modulo 998244353.

示例1

输入

2 2
ab

输出

12

说明

For the first sample, the substrings of "ab" are: "" (empty string), "a", "b", and "ab".

(1) "" + "" = ""

(2) "" + "a" = "a"

(3) "" + "b" = "b"

(4) "" + "ab" = "ab"

(5) "a" + "" = "a"

(6) "a" + "a" = "aa"

(7) "a" + "b" = "ab"

(8) "a" + "ab" = "aab"

(9) "b" + "" = "b"

(10) "b" + "a" = "ba"

(11) "b" + "b" = "bb"

(12) "b" + "bab" = "bab"

(13) "ab" + "" = "ab"

(14) "ab" + "a" = "aba"

(15) "ab" + "b" = "abb"

(16) "ab" + "ab" = "abab"

Among the sixteen strings above, only twelve are distinct："","a","b","ab","aa","aab","ba","bb","bab","aba","abb","abab"
