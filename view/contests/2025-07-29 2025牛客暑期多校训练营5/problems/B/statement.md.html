B. Extra Training

题目描述

Pigeon aspires to become a string master. Every day he trains extra-hard on string problems.

Today he is solving a problem that goes like this:

Given a string S \textstyle S S of length n \textstyle n n. You need to perform the following operation exactly k \textstyle k k times:

For the i-th operation ( 1 ≤ i ≤ k ) \textstyle (1 \leq i \leq k) (1≤i≤k)，Choose a substring of S \textstyle S S as S i \textstyle S_i Si​. S [ l i , r i ] \textstyle S[l_i,r_i] S[li​,ri​] is substring of S \textstyle S S. (i.e., the contiguous substring of S \textstyle S S starting at position l i \textstyle l_i li​ and ending at position r i \textstyle r_i ri​ , where 1 ≤ l i ≤ r i ≤ n \textstyle 1 \leq l_i \leq r_i \leq n 1≤li​≤ri​≤n). In additonal, the empty string is also a substring of S \textstyle S S.

Concatenate the chosen substrings S i ( 1 ≤ i ≤ k ) \textstyle S_i(1\leq i \leq k) Si​(1≤i≤k) in order to form a new string S 1 + S 2 + . . . + S k \textstyle S_1+S_2+...+S_k S1​+S2​+...+Sk​.

Let  X \textstyle X X be the number of distinct strings that can be formed by concatenating  k \textstyle k k substrings in this way. Output  X \textstyle X X mod 998244353.

输入描述

The first line contains two integers  n \textstyle n n and  k \textstyle k k ( 1 ≤ n ≤ 5 × 10 5 , 1 ≤ k ≤ 10 9 \textstyle 1\leq n \leq 5 \times 10^5, 1\leq k \leq 10^9 1≤n≤5×105,1≤k≤109).

The second line contains a string  S \textstyle S S of length  n \textstyle n n, only consisting of both uppercase and lowercase English letters.

Note that uppercase and lowercase letters are treated as distinct.

输出描述

Output a single integer, the number of distinct strings that can be formed by concatenating k \textstyle k k substrings (selected in order from the 1st to the k-th operation) modulo 998244353.

示例1

输入

2 2
ab

输出

12

说明

For the first sample, the substrings of "ab" are: "" (empty string), "a", "b", and "ab".

(1) "" + "" = ""

(2) "" + "a" = "a"

(3) "" + "b" = "b"

(4) "" + "ab" = "ab"

(5) "a" + "" = "a"

(6) "a" + "a" = "aa"

(7) "a" + "b" = "ab"

(8) "a" + "ab" = "aab"

(9) "b" + "" = "b"

(10) "b" + "a" = "ba"

(11) "b" + "b" = "bb"

(12) "b" + "bab" = "bab"

(13) "ab" + "" = "ab"

(14) "ab" + "a" = "aba"

(15) "ab" + "b" = "abb"

(16) "ab" + "ab" = "abab"

Among the sixteen strings above, only twelve are distinct："","a","b","ab","aa","aab","ba","bb","bab","aba","abb","abab"
