<h4>说明</h4>
<p>
  For the first sample, the substrings of "ab" are: "" (empty string),
  "a", "b", and "ab".
</p>
<p>(1) "" + "" = ""</p>
<p>(2) "" + "a" = "a"</p>
<p>(3) "" + "b" = "b"</p>
<p>(4) "" + "ab" = "ab"</p>
<p>(5) "a" + "" = "a"</p>
<p>(6) "a" + "a" = "aa"</p>
<p>(7) "a" + "b" = "ab"</p>
<p>(8) "a" + "ab" = "aab"</p>
<p>(9) "b" + "" = "b"</p>
<p>(10) "b" + "a" = "ba"</p>
<p>(11) "b" + "b" = "bb"</p>
<p>(12) "b" + "bab" = "bab"</p>
<p>(13) "ab" + "" = "ab"</p>
<p>(14) "ab" + "a" = "aba"</p>
<p>(15) "ab" + "b" = "abb"</p>
<p>(16) "ab" + "ab" = "abab"</p>
<p>
  Among the sixteen strings above, only twelve are
  distinct："","a","b","ab","aa","aab","ba","bb","bab","aba","abb","abab"
</p>
