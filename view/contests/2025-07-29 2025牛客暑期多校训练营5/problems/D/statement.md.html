D. Prime XOR Permutation

题目描述

Given an integer N \textstyle N N, you need to construct a permutation P \textstyle P P of integers from 0 \textstyle 0 0 to N − 1 \textstyle N-1 N−1 such that for all 1 ≤ i < N \textstyle 1 \leq i < N 1≤i<N, P i ⊕ P i + 1 \textstyle P_i \oplus P_{i+1} Pi​⊕Pi+1​ is a prime number. Here, ⊕ \textstyle \oplus ⊕ denotes the bitwise XOR operation.

输入描述

The first line contains an integer T \textstyle T T ( 1 ≤ T ≤ 2 × 10 5 \textstyle 1 \leq T \leq 2\times 10^5 1≤T≤2×105), the number of test cases.

For each test case, there is a single integer N \textstyle N N ( 1 ≤ N ≤ 10 6 \textstyle 1 \leq N \leq 10^6 1≤N≤106).

It is guaranteed that the sum of N \textstyle N N across all test cases does not exceed 10 6 \textstyle 10^6 106.

输出描述

For each test case:

  • If a valid permutation exists, output a single line containing N \textstyle N N space-separated integers representing the permutation P \textstyle P P.
  • If no such permutation exists, output − 1 \textstyle -1 −1.

示例1

输入

2
4
5

输出

3 1 2 0
4 1 3 0 2
