K. Perfect Journey

题目描述

In a country with n \textstyle n n cities, there are n − 1 \textstyle n-1 n−1 bidirectional roads connecting these cities, forming a tree.You are now visiting this country, and there are m \textstyle m m specific roads that you definitely want to travel through.The travel agency offers k \textstyle k k optional tour routes. Each route starts from city s i \textstyle s_i si​ and follows the shortest path to reach city t i \textstyle t_i ti​.

Your goal is to select as few routes as possible from these k \textstyle k k tour routes, ensuring that all m \textstyle m m key roads are traveled at least once.

Please calculate the minimum number of tour routes you need to select, and the number of possible ways to achieve this minimum, with the answer modulo 998244353 \textstyle 998244353 998244353. A plan is defined as the set of tour routes you choose. Two plans are considered different if and only if there exists a tour route that is selected in one plan but not in the other.

If it's impossible to travel through all the specific roads, output − 1 \textstyle -1 −1.

The problem guarantees that the answer is non-zero modulo 998244353 \textstyle 998244353 998244353.

输入描述

The first line contains three integers 2 ≤ n ≤ 2 × 10 5 \textstyle 2\leq n\leq 2\times 10^5 2≤n≤2×105, 1 ≤ m ≤ 22 \textstyle 1\leq m\leq 22 1≤m≤22, 1 ≤ k ≤ 2 × 10 5 \textstyle 1\leq k\leq 2\times 10^5 1≤k≤2×105, representing the number of cities, the number of specific roads, and the number of tour routes.

The next n − 1 \textstyle n-1 n−1 lines each contain two integers 1 ≤ u i , v i ≤ n \textstyle 1\leq u_i,v_i\leq n 1≤ui​,vi​≤n, guaranteeing that the given graph is a tree.

The next line contains m \textstyle m m distinct integers 1 ≤ x i ≤ n − 1 \textstyle 1\leq x_i\leq n-1 1≤xi​≤n−1 representing the indices of the specific roads (according to the input order).

The next k \textstyle k k lines each contain two integers 1 ≤ s i , t i ≤ n \textstyle 1\leq s_i,t_i\leq n 1≤si​,ti​≤n, indicating that the tour route goes from s i \textstyle s_i si​ to t i \textstyle t_i ti​.

输出描述

Output the minimum number of tour routes you need to select, and the number of ways to achieve this minimum, with the answer modulo 998244353 \textstyle 998244353 998244353.

If it's impossible to travel through all the specific roads, output − 1 \textstyle -1 −1.
