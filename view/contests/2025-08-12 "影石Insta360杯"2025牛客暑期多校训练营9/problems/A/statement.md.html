A. AVL tree

题目描述

Riceshower is an umamusume in the training center, and she is having a data structure class!

In the data structures class, she learned about AVL trees. An AVL tree is a type of binary search tree that is balanced based on its height. The height of a binary tree T \textstyle \mathcal{T} T is defined as follows:

  • If T \textstyle \mathcal{T} T is empty, then h T = 0 \textstyle h_\mathcal{T} = 0 hT​=0;
  • Otherwise, let the root of T \textstyle \mathcal{T} T be u \textstyle u u, and let l s u \textstyle ls_u lsu​ and r s u \textstyle rs_u rsu​ represent the left and right subtrees of u \textstyle u u (which may be empty). In this case, h T = max ⁡ ( h l s u , h r s u ) + 1 \textstyle h_\mathcal{T} = \max(h_{ls_u}, h_{rs_u}) + 1 hT​=max(hlsu​​,hrsu​​)+1.

A rooted binary tree T \textstyle \mathcal{T} T is called an AVL tree if and only if:

  • T \textstyle \mathcal{T} T is empty, or
  • Let the root of T \textstyle \mathcal{T} T be u \textstyle u u, and both the left subtree l s u \textstyle ls_u lsu​ and the right subtree r s u \textstyle rs_u rsu​ (which may be empty) of u \textstyle u u are AVL trees, and ∣ h l s u − h r s u ∣ ≤ 1 \textstyle |h_{ls_u} - h_{rs_u}| \leq 1 ∣hlsu​​−hrsu​​∣≤1.

Now, Riceshower is given a binary tree with root 1 \textstyle 1 1. However, it may not be an AVL tree. She hopes to transform the tree into an AVL tree. She can perform the following operations any number of times:

  • Delete a leaf.
  • Choose a vertex whose left son is empty and create a new vertex as its left son.
  • Choose a vertex whose right son is empty and create a new vertex as its right son.

She wants to know the minimum number of operations needed to make the tree an AVL tree.

输入描述

Each test contains multiple test cases. The first line contains the number of test cases T   ( 1 ≤ T ≤ 10000 ) \textstyle T~(1 \le T \le 10000) T (1≤T≤10000).

For each testcase, the first line contains one integer n   ( 1 ≤ n ≤ 2 × 10 5 ) \textstyle n~(1 \le n \le 2 \times 10^5) n (1≤n≤2×105), the number of vertices of the tree.

The i \textstyle i i-th of the next following n \textstyle n n lines contains two integers l s i , r s i   ( 0 ≤ l s i , r s i ≤ n ) \textstyle ls_i, rs_i~(0 \le ls_i, rs_i \le n) lsi​,rsi​ (0≤lsi​,rsi​≤n), representing the left son and the right son of vertex i \textstyle i i. If the left son or right son is empty, it is represented by 0 \textstyle 0 0.

It is guaranteed that the given tree is a binary tree with root 1 \textstyle 1 1, and ∑ n ≤ 2 × 10 5 \textstyle \sum n \le 2 \times 10^5 ∑n≤2×105.

输出描述

For each testcase, output one integer -- how many operations at least she needs to make this tree an AVL tree.
