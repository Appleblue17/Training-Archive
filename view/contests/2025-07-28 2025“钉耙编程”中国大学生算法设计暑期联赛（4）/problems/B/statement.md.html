请注意本题特殊的空间限制。

"生命野蛮蓬勃，大地千变万化。人造机器轰鸣运作，源石能的光辉照亮了暗影。人们挣扎在暗处，向往光明，在辉煌时轻视阴影，如是反复。"

对于第 i i i（ 1 ≤ i ≤ q 1\le i\le q 1≤i≤q）次询问，求满足 j − v j ≤ y i ≤ j ≤ j + v j ≤ z i j-v_j\le y_i\le j\le j+v_j\le z_i j−vj​≤yi​≤j≤j+vj​≤zi​ 的整数 j j j（ 1 ≤ j ≤ n 1\le j\le n 1≤j≤n）中， j + v j j+v_j j+vj​ 的最大值。如果 y i > z i y_i>z_i yi​>zi​ 或者不存在符合条件的整数，答案为 0 0 0。
